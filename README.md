### Authentika

Super Auth APP!

### Prerequisites
Tested with following node versions:
- 9.5.0 

## Getting Started

```
git clone https://github.com/Boffka/authentika.git
cd authentika
npm setup
npm start
```

## API Url

```
http://localhost:3000/api/v1
```
## Swagger Documentation Url

```
http://localhost:3000/explorer
```
